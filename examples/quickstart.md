# Quickstart Examples

## Save a web article
```bash
mp save https://example.com/article
```

## Save an X post
```bash
mp save https://x.com/user/status/1234567890
```

## Search all memory
```bash
mp search "why did we change auth"
```

## Search only saved links
```bash
mp find "agent workflow"
```

## Enrich the knowledge graph
```bash
mp graph enrich
mp graph stats
mp graph query King
```

## List archived content
```bash
mp list
mp list articles
mp list tweets
```
